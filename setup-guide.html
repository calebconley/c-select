♬ Data Setup

Connect Your Google Sheet

Follow these steps once and your guide updates automatically every time you edit the sheet. No code required after setup.

1

Import the CSV into Google Sheets

Open Google Sheets and create a new spreadsheet. Name it The © Select.

Go to File → Import → Upload and upload the file c-select-detroit-spots.csv that came with this package. Import it into the current sheet.

Rename the tab at the bottom to exactly: Detroit

Important: The tab name must match exactly — capital D, no spaces before or after. The guide reads from a tab named "Detroit".
2

Confirm the column structure

Your sheet should have these 6 columns in this exact order. Don't rename or reorder them.

ColumnHeaderWhat goes hereExample
ANameThe spot's nameLeila
BCategoryMust match exactly: Breakfast, Lunch, Dinner, Bar, Cafes, Hotel Lobbies, Sports Bars, Golf, Parties, Coworking, Attractions, ShoppingDinner
CTop PickTRUE or FALSE — top picks float to the top of each sectionTRUE
DBadgeShort label shown on the card. Leave blank for none.Top Pick
EMy TakeYour personal description. Keep it one sentence.The best dinner in Detroit right now.
FTagsComma-separated. 2–3 max.Middle Eastern, Must Reserve
3

Make the sheet publicly readable

The guide fetches your sheet data — so the sheet needs to be accessible. Click Share (top right), then Change to anyone with the link, set it to Viewer.

Viewer only — nobody can edit your sheet, they can only read it. Your data stays yours.
4

Copy your Sheet ID

Look at your browser's address bar when the sheet is open. The URL looks like this:

https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit

The long string between /d/ and /edit is your Sheet ID. Copy it.

5

Paste it into the guide

Open detroit-guide.html in a text editor. Near the top of the <script> section, find this line:

const SHEET_ID = 'YOUR_SHEET_ID_HERE';

Replace YOUR_SHEET_ID_HERE with your actual Sheet ID:

const SHEET_ID = '1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms'; // your real ID

Save the file. That's it.

6

Test it

Open index.html in your browser and click Detroit. The guide should load your spots from the sheet. You'll see a small status indicator in the top right of the masthead:

● live · 167 spots — Sheet is connected and loading correctly.

● local data — Sheet ID not yet set, using built-in fallback data.

● offline · local data — Sheet couldn't be reached, falling back. Check that sharing is set to public.
♬ Day-to-Day Updates

How to add or edit a spot

Add a spot: Open your Google Sheet, scroll to the bottom of the relevant category, add a new row. Fill in all 6 columns. Reload the guide — it's live.

Edit a take: Click the cell in column E and type your new description. Save. Done.

Promote to Top Pick: Change column C from FALSE to TRUE. The spot will float to the top of its category automatically.

Remove a spot: Delete the row from the sheet. It disappears from the guide on next load.

Tip: The guide fetches fresh data every time someone opens the Detroit page. No rebuilding, no re-uploading. Edit the sheet, and anyone who opens the guide next sees the update.

You're set. ♬
